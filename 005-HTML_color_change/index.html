Use button and script to change block background color to random color

00
  1. 使用256*Math.random()取得0~255之間的亂數，再用Math.floor()無條件退位取整數
  2. 重複上面的步驟3次，取得3個不同的RGB值
  3. 將取得的RGB值組合成html看得懂的rgb色碼:rgb(valueR,valueG,valueB)
  4. 更換上方色塊的顏色

  1. chrome,opera,firefox的onchange只會在滑鼠鬆開的瞬間才會動作，若要連續性的動作需用oninput
