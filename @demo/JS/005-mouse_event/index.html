onmouseenter

onmouseover

onmousemove

Note

觸發條件:
  • (靈敏度最低)onmousementer:
    游標從元素外移到目標元素內的瞬間觸發，移到子元素裡不會觸發
  • (靈敏度中等)onmouseover:
    游標從元素外移到目標元素內或 從該移到任一子元素內的瞬間觸發
  • (靈敏度最高)onmousemove:
    游標在DOM內移動時觸發
js 
function btn05enter() {
    iE += 1;
    $("#demo5enter").text("onmouseenter:" + iE);
}
function btn05move() {
    iM += 1;
    $("#d5move").text(" ,onmousemove:" + iM);
}
function btn05over() {
    iO += 1;
    $("#d5over").text(" ,onmouseover:" + iO);
}

onmouseleave

onmouseout

Note

觸發條件:
  • (靈敏度最低)onmousemleave:
    游標從目標元素內的移到元素外瞬間觸發，只要未離開目標元素都不會觸發
  • (靈敏度中等)onmouseout:
    游標從目標元素內移到元素外或 從任一子元素內移出(未超出目標元素)的瞬間觸發
