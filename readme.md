## 變數

一般的變數宣告長這樣:

```
var 變數名稱 = 變數值 ;
```
如上所示，包含了宣告方式(var)，變數名稱與變數值

#### 宣告方式  

JS的變數宣告有3種:

>**var**:可修改值的全域變數  
**let**:可修改值的區域變數  
**const**:不可修改值的全域常數
