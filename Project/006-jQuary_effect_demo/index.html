  1. jQuary語法: $(selector).action()
  2. $(document).ready(function(){ //your_code }) 當頁面中所有的DOM都載入完畢後才執行your_code
    實際使用時可縮寫為$(function(){ //your_code })即可

    3. 範例

    // 標準寫法
$(document).ready(function(){
    //your_code 
});

// 縮寫
$(function(){
    //your_code
    });
  3. 可以使用<script src="">讀取外部JS，但必須在載入外部JS前必須先載入jQuary，否則外部JS會無法運作

特效--直接顯示隱藏

顯示效果

從DOM右下角往左上角收合/展開

method:

  • .show()讓DOM顯示
  • .hide()讓DOM隱形
  • .toggle切換DOM的顯示狀態(顯示→隱藏，反之亦然)

參數

.show(speed,callback())

  • speed:執行的速度
  • callback:動作完成後執行的副程式

範例

//使用jQuary
$("#btnShow").click(function (300,"swing") {
    $("#bk101").show();
});
$("#btnHide").click(function (300,"swing") {
    $("#bk101").hide();
});
$("#btnToggle").click(function () {
    $("#bk101").toggle(300, "swing");
});

特效-淡入淡出顯示隱藏

顯示效果

顯示:快速地讓出現外框後，再將內容從透明變實體

method

  • .fadeIn()顯示DOM(透明>>實體)
  • .fadeOut()消失(實體>>透明)
  • .fadeToggle()狀態切換
  • .fadeTo()修改目標的透明度(實體>>半透明)
    注意:透過fadeTo修改過的元素若要恢復原來的透明度,須重新該DOM的style.opacity值為1(可另外寫在callback裡)

參數

  • 前三項與上面相同
  • .fadeTo(speed,opacity,callback)
    • speed:與上面相同
    • opacity:變化後的透明度[0(透明)~1(實體)]
    • callback:與上面相同

範例


// fadeIn
$("#btn201").click(function () {
    $("#bk2Code").fadeIn(1000);
});

// fadeOut
$("#btn202").click(function () {
    $("#bk2Code").fadeOut(1000);
});

// fadeToggle
$("#btn203").click(function () {
    $("#bk2Code").fadeToggle(1000);
});

// fadeTo
$("#btn204").click(function () {
    var sp=300;
    // alert($("#bk2Code").css("opacity"));
    if ($("#bk2Code").css("opacity") <1) {
        $("#bk2Code").fadeTo(sp, 1);
    }else{
        $("#bk2Code").fadeTo(sp, 0.5);
    }
});

特效--滑動顯示隱藏

顯示效果

由上往下展開元素

method

  • .slideDown()滑動顯示
  • .slideUp()滑動消失
  • .slideToggle()滑動變換狀態

參數

與上面相同

範例

$("#btn301").click(function () {
    $("#bk3Code").slideDown();
});
$("#btn302").click(function () {
    $("#bk3Code").slideUp();
});
$("#btn303").click(function () {
    $("#bk3Code").slideToggle();
});
