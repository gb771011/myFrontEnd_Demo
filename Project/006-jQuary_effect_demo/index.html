使用函式/函式庫

  1. jQuary
  2. Google code prettify
  1. jQuary語法: $(selector).action()
  2. $(document).ready(function(){ //your_code }) 當頁面中所有的DOM都載入完畢後才執行your_code
    實際使用時可縮寫為 $(function(){ //your_code })
  3. 可以使用<script src="">讀取外部JS，但必須在載入外部JS前必須先載入jQuary，否則外部JS會無法運作

簡單顯示/隱藏

效果說明

  • .show()讓DOM顯示
  • .hide()讓DOM隱形
  • .toggle切換DOM的顯示狀態(顯示→隱藏，反之亦然)

參數

範例

//使用jQuary
$("#btnShow").click(function (300,"swing") {
    $("#bk101").show();
});
$("#btnHide").click(function (300,"swing") {
    $("#bk101").hide();
});
$("#btnToggle").click(function () {
    $("#bk101").toggle(300, "swing");
});

//原生JS實現.show()
function show(targetElement) {
    var target = document.getElementsByTagName(targetElement);
    for (var i = 0; i < target.length; i++) {
        target[i].style.visibility = "visible";
    }
}
document.getElementById("show").addEventListener("click", function () { show("p"); });
//原生JS實現.hide()
function hide(targetElement) {
    var target = document.getElementsByTagName(targetElement);
    for (var i = 0; i < target.length; i++) {
        target[i].style.visibility = "hidden";
    }
}
document.getElementById("hide").addEventListener("click", function () { hide("p"); });
